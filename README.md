# A Dictionary of Color Combinations by Sanzo Wada

[Sanzo Wada](https://en.wikipedia.org/wiki/Sanzo_Wada) (1883â€“1967) was an artist, teacher, and designer. In 1927 he founded the Japan Standard Color Association (present-day Japan Color Research Institute). During the 1930s he created a six volumne book work *A Dictionary of Color Combinations*, a collection of 348 of curated colour combinations.

This website gathers together the contents of this work and reinterprets it as an interactive and functional tool.

## Installation and development instructions

It is encouraged to use **pnpm**

```bash
pnpm install
```

### 3. Run the development server

You can start the server using this command:

```bash
pnpm dev
```
